Programming Language

컴퓨터에게 명령이나 연산을 시킬 목적으로 설계되어 기계와 의사소통을 할 수 있게 해주는 언어를 뜻한다.
이를 이용해 사람이 원하는 작업을 컴퓨터가 수행할 수 있도록 프로그래밍 언어로 일련의 과정을 작성하여 일을 시킨다.
우리가 작성하는 언어들은 사람이 이해하기 쉽도록 만든 언어로 실행전에 컴파일러를 통해서 컴퓨터가 이해할수 있는 기계어로 컴파일된 후 실행된다.
이 프로그래밍 언어를 편집하기 위해 예전에는 각각의 개발환경에 맞추어서 다른형태의 툴로 존재했지만 현 시점에서 대부분의 소프트웨어 개발툴들은
개발에 필요한 대부분의 요소가 통합되어 있는 형태로 배포된다.
이것을 통합 개발 환경 라고 부르며 IDE라고 부르기도 한다.
대표적인 언어로 C언어, JavaScript, HTML 등이 있다.
